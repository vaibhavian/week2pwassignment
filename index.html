Full Stack Web Development Tech Stack

Front-end Development
Front-end development involves creating the user interface and user experience of a website or web application. It typically includes HTML, CSS, and JavaScript to build responsive, interactive, and visually appealing interfaces that users interact with directly in their browsers.
Back-end Development
Back-end development focuses on the server-side of web applications. It involves working with databases, server logic, and APIs to handle data storage, processing, and server-side operations. Common back-end languages include Python, Ruby, Java, Node.js, and PHP.
Database Management
Database management refers to the process of designing, implementing, and maintaining databases to store and manage application data efficiently. Common database systems used in web development include MySQL, PostgreSQL, MongoDB, and SQLite.
Version Control
Version control systems (VCS) allow developers to track changes to their code over time. It enables collaboration, rollback to previous versions, and avoids conflicts when multiple developers are working on the same project. Git is the most widely used version control system.
Deployment & Hosting
Deployment and hosting involve making a web application accessible to the public on the internet. It includes configuring servers, setting up environments, and deploying code. Common hosting platforms include AWS, Heroku, DigitalOcean, and VPS providers.

Question 9

  1. <em> - Emphasis Tag
    • Use to emphasize text, typically rendered as italic.
    • Used to highlight keywords, idiomatic phrases, etc.
    • Don't use for styling alone; use CSS for visual emphasis.
  2. <strong> - Strong Tag
    • Used to give strong importance to text, typically rendered as bold.
    • Use for critical or important content.
    • Don't use for styling alone; use CSS for visual emphasis.
  3. <mark> - Mark Tag
    • Used to highlight or mark specific text within a paragraph.
    • Commonly used in search results or to indicate relevance.
    • Don't use for permanent formatting; it's for temporary highlighting.
  4. <sub> - Subscript Tag
    • Used to display subscripted text, such as chemical formulas (H2O).
    • Use sparingly, as overuse can affect readability.
  5. <sup> - Superscript Tag
    • Used to display superscripted text, such as mathematical exponents (x2).
    • Use sparingly, as overuse can affect readability.
